## Pasos del Proceso — Servicio Recurrente

### Paso 1 — Reclasificación de Trabajadores
- **Responsable:** Germain
- **Descripción:** Se identifican los trabajadores elegibles del cliente para ser reclasificados de empleados a contratistas. Cada trabajador reclasificado debe completar un formulario W-9.
- **Entregable:** W-9 completado y archivado por cada contratista elegible.

> **PREGUNTA ABIERTA (discutir en próxima reunión):** ¿Cuál es el criterio para determinar si un trabajador califica como empleado o contratista? Se necesita establecer una política clara antes de ejecutar este paso.

### Paso 2 — Almacenamiento de Datos para Formularios 1099
- **Responsable:** Germain
- **Descripción:** Los datos recopilados de los W-9 se almacenan en la base de datos interna para preparar el Form 1099-NEC o 1099-MISC al cierre del año fiscal. Los formularios son enviados a cada contratista al finalizar el año.
- **Entregable:** Base de datos de contratistas actualizada. Formularios 1099 enviados a fin de año.

### Paso 3 — Reunión Inicial de Estrategia Fiscal (Interna)
- **Responsable:** Abner y Germain
- **Descripción:** Al incorporarse el cliente, Abner y Germain se reúnen internamente para analizar la estructura contable del cliente e identificar posibles estrategias para reducir la carga fiscal. Esta reunión se realiza una sola vez al inicio del servicio.
- **Entregable:** Estrategias fiscales identificadas y documentadas internamente.

### Paso 4 — Reunión Trimestral de Estrategia Fiscal
- **Responsable:** Abner y Germain
- **Descripción:** Cada trimestre, Abner y Germain revisan la situación fiscal del cliente, evalúan el impacto de las estrategias aplicadas y ajustan según sea necesario.
- **Entregable:** Notas de reunión trimestral guardadas en GHL. Estrategias actualizadas si aplica.

### Paso 5 — Mantenimiento Mensual de Base de Datos de Trabajadores
- **Responsable:** Germain
- **Descripción:** Mensualmente se revisa y actualiza la base de datos de trabajadores/contratistas del cliente. Se elimina a quienes ya no prestan servicios y se incorpora a los nuevos, solicitando su W-9 correspondiente.
- **Entregable:** Base de datos de contratistas actualizada cada mes.

## Variante: Servicio de Única Vez — Tax Filing por Deadline

### Descripción
Para clientes que solo requieren asistencia puntual para presentar sus impuestos antes del deadline de temporada fiscal.

### Pasos

#### Paso 1 — Organización del Bookkeeping
- **Responsable:** Germain y su equipo
- **Descripción:** El equipo de Germain pone en orden todos los registros contables del cliente necesarios para la presentación de impuestos.
- **Entregable:** Libros contables del cliente organizados y listos.

#### Paso 2 — Envío de Documentos a la Firma Contable
- **Responsable:** Germain
- **Descripción:** Germain envía los documentos financieros del cliente a la firma contable tercera para que realice el filing de impuestos.
- **Entregable:** Documentos enviados a la firma contable.

#### Paso 3 — Extensión (si aplica)
- **Responsable:** Germain
- **Descripción:** Si el tiempo disponible no es suficiente para completar el proceso antes del deadline, Germain gestiona la solicitud de una extensión ante las autoridades fiscales correspondientes.
- **Entregable:** Extensión solicitada y confirmada (si aplica).

#### Paso 4 — Confirmación de Filing
- **Responsable:** Germain
- **Descripción:** Una vez que la firma contable confirma la presentación de los impuestos, Germain notifica al cliente y cierra el servicio en GHL.
- **Entregable:** Cliente notificado. Servicio cerrado en GHL.
